BottomBarSharp

  • Xamarin
  • Android
  • C#
  • Library
  • Github

このライブラリはBottomBarをC#に翻訳したものです。
公式のサポートライブラリに同じコンポーネントが含まれるようになったのでそちらを利用したほうがいいです。

著作について

BottomBarはApache License 2.0に基づいて配布されています。
Copyright (c) 2016 Iiro Krankka (http://github.com/roughike).

このライブラリについて

  • Xamarin.Android
  • BottomBar 2.0.2を翻訳しています
  • NuGetで公開していません
  • 使用する場合は各自ビルドする必要があります

BottomBarからの変更点

  • xmlに書く名前空間が変更されています(com.roughike.bottombar.BottomBar → bottombarsharp.BottomBar)
  • タブの選択・再選択のEventHandlerも用意しています
  • いくつかのフィールドをプロパティとして用意しました

使用方法

  1. ソリューションをPCにダウンロードまたはgitでclone
  2. Xamarinに開発環境を含んだVisual Studio 2015でこのソリューションを開く
  3. ソリューションをビルドする
  4. ソリューションをファイルエクスプローラーで開きbinフォルダ内にあるdllファイルを探す
  5. 使用したいプロジェクトでそのdllファイルを参照する

サンプル

BottomBar.Appプロジェクトを見てください。

トラブルシューティング

リソースが見つからない

リソースIDに大文字を含めて指定している場合リソースが見つからないことがあるようです、小文字で指定してみてください。
例) @drawable/IconButtonGrey24dp → @drawable/iconbuttongrey24dp
