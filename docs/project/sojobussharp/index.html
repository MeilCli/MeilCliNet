ソースコードについて

このアプリのソースコードはMVVMパターンに基づいて作成されています、これはクロスプラットフォームな開発に適したデザインパターンとなっています。
各ビルドターゲット向けのソースは基本的にはViewを用意するだけとなります、ViewModelについてはBusViewModel.csを確認してください。

また、ライセンス表示についてはLicenseViewModel.csを使用するといいです、追加のライセンス表示がある場合はLicenseViewModelクラスを継承しInitLicenseメソッドをオーバーライドしてください。
